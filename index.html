A fullstack React.js web application that scrapes YouTube with the YouTube Data API, finds new trailers, and adds them to a PostgreSQL database via a Node.js server and Express API.

Each trailer is parsed and has movie poster and release date information added to its Database entry using the TMDB API.

Trailer information after each scrape interval is cached using a Redis cache on the Heroku hosting service to expediate client requests, and avoid unnecessary extra calls to the database.

OAuth is used for user authentication, allowing users to mark spoiler sections in videos. These sections are then attached to the trailers allowing other users to have these sections blocked while watching, as well as vote on user submitted content. User data is stored in a global state using Redux.

There is also an admin backend which allows administrators to manually update trailer information. Admin accounts are secured using JSON Web Token authentication on the server.
